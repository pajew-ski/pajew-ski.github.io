# Digitales Nullpunkt-Feld

A high-performance, minimalist web presence ("Premium Experience") built with:

- **React 19**
- **Vite**
- **TypeScript**
- **Tailwind CSS** (Premium Monochromatic Theme)
- **Framer Motion** (Animations & Interactions)
- **Three.js / React Three Fiber** (WebGL Hero Interaction)
- **i18next** (English/German Support)

## Architecture

- **Halo-Effekt / First Impression Engineering:** Interactive WebGL particle network (`ParticleNetwork.tsx`) that visualizes "Negentropy" (order restoration) on user interaction.
- **Kognitive Fluenz:** Minimalist, monochromatic design with high whitespace and clear hierarchy.
- **Peak-End-Regel:** Subtle animations and hover states.
- **Micro-Interactions:** Smooth scrolling, responsive elements.

## Development

```bash
npm install
npm run dev
```
